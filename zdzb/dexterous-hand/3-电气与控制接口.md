    <h2>电气与控制接口</h2>
    
    <h3>电气参数</h3>
    <ul>
        <li>工作电压：DC 24V ±10%</li>
        <li>静态电流：0.2A</li>
        <li>最大工作电流：2A</li>
        <li>工作温度：0℃ ~ 50℃</li>
    </ul>
    
    <h3>接线说明</h3>
    <p>灵巧手的接线接口位于手掌背面，包含电源接口和通讯接口。以下是详细的接线说明：</p>

    <p><strong>接线注意事项：</strong></p>
    <ul>
        <li>请确保使用符合规格的电源适配器（DC 24V ±10%）</li>
        <li>接线前请确保电源已关闭，避免短路</li>
        <li>按照接口标识正确连接线缆，不要强行插拔</li>
        <li>定期检查接线是否松动，确保连接稳定</li>
    </ul>
    
    <h3>通讯接口</h3>
    <ul>
        <li><strong>主接口</strong>：CAN / BLE</li>
        <li><strong>功能支持</strong>：支持获取舵机参数，支持获取触碰数据</li>
        <li><strong>协议文档</strong>：相关通讯协议详见专门的通讯协议文档</li>
    </ul>
    <p>通讯接口的设计考虑了不同的应用场景需求，CAN接口适用于工业环境，提供稳定的长距离通讯能力；BLE接口适用于短距离无线控制，便于调试和移动应用。</p>
    
    <h3>控制模式</h3>
    <ul>
        <li><strong>位置控制</strong>：精确控制手指位置，实现精确抓取</li>
        <li><strong>力控/力矩控制</strong>：通过力反馈进行控制，确保操作的安全性</li>
        <li><strong>手势复现</strong>：能够复现预设的手势动作</li>
        <li><strong>动捕手套遥操</strong>：支持通过动捕手套进行远程操控</li>
    </ul>
    <p>这些控制模式提供了灵活的控制方式，适应不同的应用场景。无论是自动化的精确控制，还是通过动捕手套进行的手动操作，都能满足用户的需求。</p>
    
    <h3>控制精度</h3>
    <ul>
        <li>位置控制精度：0-255</li>
        <li>力矩控制精度：0-100</li>
        <li>关节控制频率：最高50 Hz</li>
    </ul>
