<h3>演示上位机</h3>
<p>本软件为灵巧手专用控制软件，提供可视化操作界面与实时状态监控。软件界面分为<strong>四大核心区域</strong>：</p>

</section>

<section class="product-section">

<h3>一、灵巧手动作控制及触碰展示区</h3>

<p>位于软件界面左上方，是灵巧手动作控制的核心操作区域，同时集成指尖触碰状态的可视化展示功能。</p>

<h4>1.1 快捷手势控制</h4>
<p>顶部手势栏提供预设手势的一键触发功能：</p>

<p>此外，手势栏还提供以下参数调节：</p>
<ul>
<li><strong>动作循环</strong>：勾选后，预设手势（除抓握外）将循环执行</li>
<li><strong>手指力度</strong>：通过「-」「+」调节，默认25%，范围0%-100%，控制动作输出力度</li>
<li><strong>手指速度</strong>：通过「-」「+」调节，默认25%，范围0%-100%，控制动作执行速度</li>
</ul>

<h4>1.2 手指姿态精细调节</h4>
<p>每根手指（大拇指、食指、中指、无名指、小拇指）均设有独立控制区域，包含：</p>
<ul>
<li><strong>侧摆滑块</strong>：控制手指向手掌左右摆动的角度</li>
<li><strong>弯曲滑块</strong>：控制手指向掌心弯曲的角度</li>
<li><strong>横摆滑块</strong>（仅大拇指）：控制大拇指横向外展/内收的角度</li>
</ul>

<h4>1.3 触碰压力可视化展示</h4>
<p>每个手指控制区域顶部设有<strong>指尖触碰状态指示器</strong>，以颜色渐变方式直观展示手指肚（指尖腹面）受到的实时压力：</p>

<p>颜色变化遵循<strong>灰 → 绿 → 蓝 → 红</strong>的渐变规律，颜色越深（越接近红色）表示指尖受到的压力越大。用户可通过颜色快速判断灵巧手与外界物体的接触力度，便于在抓取操作中实时调整力度，避免损坏被抓物体或灵巧手本体。</p>

<div class="highlight-box">
<p><strong>应用场景</strong>：在抓取易碎物品（如鸡蛋、水果）时，通过观察指尖颜色变化，可实时掌握接触力度，实现"柔顺抓取"，既保证抓取稳定又不损坏物体。</p>
</div>

<h3>二、灵巧手状态反馈区</h3>

<p>位于软件界面底部，实时显示灵巧手各关节的运行状态，为用户提供全面的设备健康监测与调试信息。</p>

<ul>

<li><strong>电流监测</strong>：异常电流可能表示关节受阻或故障</li>
<li><strong>位置对比</strong>：当前位置与目标位置的差值可判断运动完成度</li>
</ul>

<h3>三、VR动捕手套控制区</h3>

<p>位于软件界面右侧，用于连接VR手套并实现动作捕捉数据的实时同步。连接成功后即可直接遥操灵巧手，让灵巧手实时复刻人手动作。</p>

<h4>3.1 设备管理</h4>

<table class="spec-table">
<thead>
<tr>
<th>按钮/开关</th>
<th>功能说明</th>
</tr>
</thead>
<tbody>
<tr>
<td>设备下拉框</td>
<td>选择待连接的VR手套设备（扫描后显示可用列表）</td>
</tr>
<tr>
<td>扫描</td>
<td>搜索当前可用的VR手套蓝牙设备</td>
</tr>
<tr>
<td>连接/断开</td>
<td>建立或断开与VR手套的通信连接</td>
</tr>
<tr>
<td>使能CAN</td>
<td>开启后启用CAN总线，用于连接灵巧手硬件</td>
</tr>
</tbody>
</table>

<div class="highlight-box warning">
<p><strong>重要</strong>：「使能CAN」开关用于切换控制模式。关闭状态仅能用滑块控制，开启状态仅能用VR手套控制，两者不能同时使用。</p>
</div>

<h4>3.2 对指校准功能</h4>
<p>该区域同时展示VR手套的实时姿态数据（拇指横摆、拇指弯曲、各指弯曲等），并配备对指校准工具，用于进阶的对指操作校准：</p>
<ul>
<li><strong>重置</strong>：将所有手套数据清零，恢复初始状态</li>
<li><strong>手指记录按钮</strong>（食指/中指/无名指/小拇指）：分别记录对应手指对指状态时大拇指横摆姿态参数。记录的数据会显示在大拇指横摆参数中，用于实现精准的大拇指与其他手指的对指操作</li>
</ul>
<p>对于基础遥操，连接手套后可直接使用，无需校准。当需要进行精准对指等进阶操作时，再使用校准功能记录各手指的对指参数即可。</p>

</section>

<section class="product-section">

<h3>四、灵巧手孪生展示区</h3>

<p>以三维可视化方式展示灵巧手的实时姿态，形成"数字孪生"效果，让用户直观掌握灵巧手当前状态。</p>

<h4>4.1 功能说明</h4>
<ul>
<li><strong>实时姿态同步</strong>：虚拟灵巧手模型与真实灵巧手姿态实时同步，直观展示当前手部状态</li>
<li><strong>自由视角切换</strong>：支持自由拖拽、旋转、缩放，从任意角度观察灵巧手姿态，便于调试与演示</li>
</ul>

<h4>4.2 应用价值</h4>
<ul>
<li>远程操控时实时观察灵巧手状态，无需现场查看</li>
<li>调试过程中从多角度直观判断姿态是否正确</li>
<li>演示展示时提供清晰直观的视觉效果</li>
</ul>

</section>

<section class="product-section">

<h3>五、标准使用流程</h3>

<h4>第一步：硬件准备</h4>
<ol>
<li>将CAN工具插入电脑USB口</li>
<li>将XT-30插头接入灵巧手</li>
<li>确认硬件接线正常、无松动</li>
<li>连接灵巧手电源线</li>
</ol>

<h4>第二步：设备连接</h4>

<div class="feature-grid">
<div class="feature-item">
<h4>手动控制方式</h4>
<ol>
<li>点击左上角红色「CAN未连接」按钮</li>
<li>在弹出界面点击「刷新」，自动识别CAN接口工具端口</li>
<li>如未显示，可在下拉菜单手动选择对应设备</li>
<li>点击「连接」完成连接</li>
</ol>
<p>提示：先插设备再开软件，通常会自动识别并填充。</p>
</div>
<div class="feature-item">
<h4>VR手套控制方式</h4>
<ol>
<li>在右侧VR手套控制区点击「扫描」按钮</li>
<li>等待扫描完成，识别到设备会自动填充</li>
<li>若无显示，在下拉框选择手套蓝牙目标设备</li>
<li>点击「连接」建立通信</li>
<li>打开「使能CAN」开关</li>
</ol>
</div>
</div>

<div class="highlight-box warning">
<p><strong>重要</strong>：手动控制和VR手套控制方式只能选择其一。使用手动控制时使能CAN为关闭状态，使用VR手套控制则需打开使能CAN。</p>
</div>

<h4>第三步：控制操作</h4>
<ul>
<li><strong>快捷控制</strong>：点击顶部手势图标，快速触发预设手势；勾选「动作循环」可让手势重复执行</li>
<li><strong>精细控制</strong>：拖动各手指姿态滑块，实现手指侧摆、弯曲、横摆的精准调节</li>
<li><strong>手套联动</strong>：连接VR手套后，手套姿态数据实时同步至软件，控制灵巧手复刻手套动作</li>
</ul>

<h4>第四步：状态监控</h4>
<p>查看底部「手指关节姿态信息表」，实时监控各关节位置、速度、温度与电流等运行状态。</p>

<h4>第五步：断开连接</h4>
<ol>
<li>先关闭「使能CAN」开关</li>
<li>点击「断开」按钮断开VR手套（如使用）</li>
<li>关闭软件，完成操作流程</li>
</ol>

</section>

<section class="product-section">

<h3>六、注意事项</h3>

<ul>
<li>设备未连接时，所有控制功能与数据显示均为初始状态，无法执行动作</li>
<li>调节力度与速度时，建议逐步调整，避免过大参数导致硬件过载或损坏</li>
<li>若出现连接失败或异常告警，请检查硬件接线与电源，重启软件后重试</li>
<li>请勿在设备运行时强行断电，以免损坏硬件</li>
<li>定期查看关节温度与电流数据，若数值超出正常范围，应立即停止操作并排查故障</li>
<li>请勿在潮湿、高温、强电磁干扰环境下使用本系统，避免影响设备稳定性</li>
</ul>

</section>
